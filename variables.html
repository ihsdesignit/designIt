Posted by Neha Bashyam and Emily Chin on September 20, 2017

The Basics

In code, values can be saved into variables. A variable is a nickname for a value that can be used to refer to that value later. Assigning a variable means specifying what the variable is holding. Functions, on the other hand, are chunks of code that you can reuse by calling the function. Functions, like variables, must be defined before they are called.
